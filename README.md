# Rust-project-learn

Piccolo progetto didattico in Rust per imparare le basi del linguaggio.

- Scopo: sperimentare tipi, variabili, input/output e organizzazione in moduli.
- Contenuti: file in `src/` come `main.rs`, `function.rs`, `input.rs`, `types.rs`, `variable.rs`.
- Eseguire: eseguire `cargo run` dalla cartella del progetto.
- Note: codice pensato per apprendimento e sperimentazione; estendibile liberamente.
